# Setting Up Visitor Map

To add a visitor map to your website, follow these steps:

## Option 1: ClustrMaps (Recommended)

1. Visit [ClustrMaps.com](https://clustrmaps.com/)
2. Click "Get Your Free Map"
3. Enter your website URL: `https://hcwdavid.github.io`
4. Complete the registration
5. Copy the HTML code they provide
6. Open `docs/_layouts/default.html`
7. Find the section with `YOUR_MAP_ID` and replace it with the code from ClustrMaps

## Option 2: Flagcounter

1. Visit [Flagcounter.com](https://flagcounter.com/)
2. Customize your counter
3. Get the HTML code
4. Replace the visitor map section in `docs/_layouts/default.html`

## Option 3: Simple Analytics (Free alternative)

The visitor map section is already set up in your layout. Just sign up at ClustrMaps and get your tracking code!

## Current Setup

The visitor map is located at the bottom of every page in the footer section.
It will show a world map with dots indicating where your visitors are from.
